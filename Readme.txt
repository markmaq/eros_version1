English
=======

For Windows, run WD Discovery.exe to install WD software to help you get a backup plan and modify drive settings. If you don’t have internet connectivity, please install WD Backup™ and WD Drive Utilities™ from the WD Software Offline Installers > For Windows folder.

For Mac, open WD Discovery for Mac.dmg to install WD software to help modify drive settings. If you don’t have internet connectivity, please install WD Drive Utilities™ from the WD Software Offline Installers > For Mac folder.
